# 彩票分析系统规则

## 核心原则

### 1. 试机号/开机号使用规则 ⭐⭐⭐
**重要程度：最高**

- **试机号和开机号仅作为参考依据**
- **严禁将试机号或开机号直接作为开奖号码推荐**
- **精选2注必须与试机号、开机号完全不同**
- **30注大底中不应包含试机号和开机号**

#### 实现方式：
```python
# 生成推荐号码前，先定义排除列表
excluded_numbers = [today_test, today_boot]  # 试机号和开机号

# 生成精选2注时，检查并排除
def generate_recommendation(excluded):
    while True:
        nums = generate_from_hot_numbers()  # 基于热号生成
        if nums not in excluded:
            return nums
```

### 2. 数据分析基础
- 排列三：基于1000期历史数据
- 福彩3D：基于2000期历史数据
- 使用3阶马尔可夫链进行预测

### 3. 推荐策略
1. **热号组合**：从各位置热号中选择，避免与试机号/开机号重复
2. **冷热搭配**：热号+冷号组合，提高覆盖面
3. **和值跨度分析**：基于历史数据统计

### 4. 输出格式
- 精选2注（重点关注）
- 30注大底
- 和值/跨度推荐
- 形态推荐
- 位置热号分析
