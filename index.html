权重调优测试

测试过程：

• 测试了8种不同权重组合

• 最终确定最佳配置：

334断组(14%) + 五五分解(9%) + 两码和差(6%) + 012路(6%) + 奇偶(4%) + 形态(4%) + 和值尾(7%)

此配置在5000期回测中取得最高命中率13.26%

7种分析方法
334断组法 (14%)
根据上期和值尾断组，杀掉最冷的一组
五五分解法 (9%)
四种分解式：大小/奇偶/质合/四方，选热组
两码和差法 (6%)
计算两码和、差、进位，选高频数字
012路分析法 (6%)
按除3余数分类，选各位置热路
奇偶分析法 (4%)
统计近30期奇偶比例，选热类型
形态分析法 (4%)
大小+质合组合分析，选热形态
和值尾分析法 (7%)
统计近30期和值尾，选热尾对应数字

综合评分权重：334断组14% + 五五分解9% + 两码和差6% + 012路6% + 奇偶4% + 形态4% + 和值尾7%
